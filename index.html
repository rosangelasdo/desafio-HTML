FERRAMENTAS UTILIZADAS

Voltar

  • VISUAL STUDIO CODE

    • Ferramenta para desenvolvimento de páginas código-fonte HTML. Funciona em ambiente Windows e Linux. Integra com outras ferramentas, trazendo facilidade no desenvolvimento.

  • LIVE SERVER

    • Dá a facilidade de visualização da aplicação em execução no navegador a partir do Visual Studio Code.

  • INSPETOR DE ELEMENTOS

    • É uma ferramenta do navegador. Permite visualizar todo o código HTML da página. Isto facilita muito para quem está aprendendo ou quer tirar alguma dúvida sobre como foi construída a página em visualização

USANDO O INSPETOR DE ELEMENTOS

Voltar

Ao abrir uma página web, podemos acionar o "inspetor de elementos" para visualizar e compreender como foi desenvolvido o código-fonte que está por traz desta página. Isto facilita, por exemplo, para aprendizagem ou mesmo para a vida prática de desenvolvimento. Para acioná-lo, basta pressionar o botão direito do mouse e escolher "inspecionar" . Será então exibido o código-fonte da página, permitindo a navegação em todo o código-fonte. Deste modo, podemos fazer alterações e visualizá-las, porém estas não atualizam a página real.

ESTRUTURA BÁSICA DO HTML

Voltar

HTML é um arquivo de marcação de Hipertexto. Não é uma linguagem de programação. É composto por cabeçalho Head e corpo body

TAGS

Voltar

Tags são as sinalizações do HTML, as quais dão formatações desejadas a página web. Nesta aula, experimentamos algumas tags iniciais, além do editor da ferramenta Visual Studio que facilita muito a inclusão das tags.

Os atributos das tags são as propriedades a elas atribuídas, como tipo, tamanho, cores.

TEXTOS

Voltar

Equivalem a tipografia HTML. Dentre eles, vimos as tags utilizadas para cabeçalho, parágrafos e citações, bem como tags para formatação visual dos textos.
